Frontend (Vite + React suggested)

Quick start:

1. `cd frontend`
2. `npm install`
3. `npm run dev`

Suggested pages/components:

- Group list and group detail
- User dashboard (list of activities assigned to the user)
- Activity submission form (attach proofs/images)
- Review panel for group members to accept/reject submissions

This scaffold includes placeholders in `src/` to get started.
